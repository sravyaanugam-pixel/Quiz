# 🧠 General Knowledge Quiz

A simple Python-based General Knowledge Quiz game that asks users three questions, checks their answers, and displays the final score.

## 📌 Features

- Asks 3 general knowledge questions.
- Accepts user input.
- Checks answers using if-else statements.
- Keeps track of the score.
- Displays the final score at the end.
- Beginner-friendly Python project.

## 🛠️ Technologies Used

- Python 3

## 📂 Project Structure

```
General-Knowledge-Quiz/
│── quiz.py
│── README.md
```

## ▶️ How to Run

1. Install Python 3.
2. Download or clone this repository.
3. Open a terminal in the project folder.
4. Run:

```bash
python quiz.py
```
